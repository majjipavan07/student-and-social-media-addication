# 📊 Student Social Media Addiction - Data Analysis Project

## 📌 Project Overview

This project analyzes a dataset on **Student Social Media Addiction**
using Python. The goal is to understand students' social media usage
patterns, identify trends, and gain meaningful insights through data
analysis and visualization.

## 🎯 Objectives

-   Load and explore the dataset.
-   Clean and preprocess the data.
-   Perform Exploratory Data Analysis (EDA).
-   Visualize important patterns and relationships.
-   Generate insights from student social media usage.

## 🛠️ Technologies Used

-   Python 3
-   Jupyter Notebook
-   Pandas
-   NumPy
-   Matplotlib
-   Seaborn
